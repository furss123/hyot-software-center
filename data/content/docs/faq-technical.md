## System Requirements

All software requires Windows 10 22H2 or later.
Both x64 and ARM64 architectures are supported unless noted otherwise.

## Auto-start

Some utilities support auto-start on login via Windows Task Scheduler.
This is always opt-in and can be disabled in app settings.
