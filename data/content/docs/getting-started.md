## Installation

Download the installer from the software detail page and run it.
Every release includes a SHA256 checksum for verification.

## Verify Checksum (PowerShell)

```powershell
Get-FileHash .\filename.exe -Algorithm SHA256
```

Compare the output with the SHA256 shown on the download page.

## Portable Version

Extract the ZIP file to any folder and run the executable directly.
No installation required.
